# 📝 CLI To-Do List App

A simple command-line To-Do List application built using Python.

## 🚀 Features
- Add tasks
- View tasks (numbered)
- Remove tasks
- Clear all tasks
- Persistent storage using file handling

## 🛠️ Technologies Used
- Python

## 💾 How It Works
Tasks are stored in a file (`tasks.txt`) so they remain saved even after closing the program.

## ▶️ How to Run

```bash
python todo.py
```
